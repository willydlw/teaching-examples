

### Build Instructions 

- **Create the build directory** from your project's roo directory:

```bash 
mkdir build 
```

- **Change to the build directory:** 

```bash 
cd build
```

- **Run CMake to generate build system files (e.g., Makefiles):**

```bash 
cmake ..
``` 

- **Build the project:**

```bash 
cmake --build .
```
